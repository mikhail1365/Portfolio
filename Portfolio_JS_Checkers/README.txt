Игра шашки. 
Использованные технологии: HTML, CSS, JS, DOM.
Файл "script.js", отвечающий за логику написания игры, подробно прокомментирован.

Здесь реализованы следующие возможности и правила игры:
1. нельзя ходить назад;
2. если можно съесть шашку противника, ее обязательно нужно съесть;
3. при достижении последней горизонтали шашки превращаются в дамки;
4. реализованы следующие возможности:
	- начать новую игру, 
	- поменять сторону (переменная "StartMoveColor"), 
	- поменять цвет клеток, в которых расположены шашки (переменная "colorWorkPixels"), 
	- перевернуть доску (переменная "downsideColor")
5. на каждом ходу указывается, чей ход;
6. помечаются цветом клетки, куда походил противник на предыдущем ходу.
